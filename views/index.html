Exercise Tracker

POST /api/exercise/new-user

Create a New User

POST /api/exercise/add

Add exercises

GET /api/exercise/users

Get All Users

GET /api/exercise/log

Get User's Exercise Log

Examples:

/api/exercise/log?userId=12345 - Get full exercise log

/api/exercise/log?userId=12345&from=2024-01-01&to=2024-12-31 - Get log with date filter

/api/exercise/log?userId=12345&limit=5 - Get log with limit
